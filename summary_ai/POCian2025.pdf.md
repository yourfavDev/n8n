## 3. Rezolvare Pas cu Pas

### Subiectul 1: Contabilizare și Reconciliere

**Pasul 1: Calculul diferențelor temporare la 31.12.N**

Trebuie să determinăm Valoarea Contabilă (VC) și Baza Fiscală (BF) pentru activele relevante.

1.  **Instalații:**
    *   Amortizare Contabilă (anuală) = $63.000 / 9 = 7.000$.
    *   Amortizare Fiscală (anuală) = $63.000 / 7 = 9.000$.
    *   VC (la 31.12.N, după 1 an) = $63.000 - 7.000 = 56.000$.
    *   BF (la 31.12.N, după 1 an) = $63.000 - 9.000 = 54.000$.
    *   Diferență: VC > BF => Diferență Temporară Impozabilă (DTI) = $2.000$.

2.  **Dobânzi de încasat:**
    *   VC = $8.000$ (venit recunoscut).
    *   BF = $0$ (impozabil la încasare).
    *   Diferență: VC > BF => DTI = $8.000$.

3.  **Terenuri:**
    *   VC = $210.000$ (reevaluat).
    *   BF = $150.000$ (cost istoric).
    *   Diferență: VC > BF => DTI = $60.000$.

4.  **Dividende:**
    *   Diferență permanentă (nu generează impozit amânat).

**Tabel Centralizator (pentru cerința a):**

| Elemente | VC | BF | DTI | DTD |
| :--- | :--- | :--- | :--- | :--- |
| Instalații | 56.000 | 54.000 | 2.000 | - |
| Dobânzi de încasat | 8.000 | 0 | 8.000 | - |
| Terenuri | 210.000 | 150.000 | 60.000 | - |
| **Total** | | | **70.000** | **-** |

**Calcul Impozit Amânat:**
*   Datorie Impozit Amânat (DIA) finală = $70.000 \times 16\% = 11.200$.
*   DIA existentă (sold inițial) = $640$.
*   DIA de înregistrat = $11.200 - 640 = 10.560$.

Din suma de 11.200:
*   Aferent Terenuri (Capitaluri proprii): $60.000 \times 16\% = 9.600$.
*   Aferent Rezultatului (Cheltuială): $(2.000 + 8.000) \times 16\% = 1.600$.
*   Verificare: $9.600 + 1.600 = 11.200$.
*   Deoarece soldul inițial era 640 (probabil aferent rezultatului sau capitalurilor anterioare, presupunem aferent rezultatului pentru simplificare sau ajustăm global).
    *   Dacă 640 era din rezultat, cheltuiala cu impozitul amânat este $1.600 - 640 = 960$.
    *   Dacă 640 era din rezerve, ajustarea se face pe rezerve.
    *   *Presupunere:* Ajustăm soldul total 4412.
    *   Înregistrare pentru teren (pe seama capitalurilor):
        $$1034/105 = 4412 \quad 9.600$$
    *   Înregistrare pentru restul (pe seama cheltuielilor):
        *   Datorie totală necesară din P&L elemente = $1.600$.
        *   Existent = $640$. Diferență = $960$.
        $$6912 = 4412 \quad 960$$

**Pasul 2: Reconcilierea (Cerința b)**

*   Rezultat Contabil Brut (RCB) = 100.000.
*   Impozit teoretic (16%) = 16.000.
*   Deduceri fiscale suplimentare (Amortizare): $9.000 (fiscal) - 7.000 (contabil) = 2.000$ (deducere extra).
*   Venituri neimpozabile (Dividende) = 10.000.
*   Venituri neimpozabile curent (Dobânzi - impozabile în N+1) = 8.000.
*   Rezultat Fiscal = $100.000 - 2.000 (\Delta Amort) - 10.000 (Div) - 8.000 (Dob) = 80.000$.
*   Impozit curent = $80.000 \times 16\% = 12.800$.

**Verificare reconciliere:**
Cheltuiala totală cu impozitul (Curent + Amânat) = $12.800 (Curent) + 960 (Amânat P&L) = 13.760$.

*Verificare alternativă:*
Taxa teoretică ($100.000 \times 16\%$) = 16.000.
Efect dividende (neimpozabile) = $-10.000 \times 16\% = -1.600$.
Alte elemente permanente? Nu.
Total așteptat = $16.000 - 1.600 = 14.400$.
*Notă:* Aici există o discrepanță uzuală în problemele studențești între abordarea temporală și cea a contului de profit și pierdere dacă nu se specifică clar originea soldului inițial de 640. Dacă ignorăm soldul inițial și calculăm impactul pe an:
Impact profit curent: Amortizare (deductibilă temp) - reduce impozit curent, crește amânat. Dobânzi (neimp temp) - reduce impozit curent, crește amânat.
Total Impozit P&L = Impozit Curent (12.800) + Cheltuială Impozit Amânat (generată în an: 1.600) = 14.400.
Aceasta corespunde cu $16.000 - 1.600$.
Deci, cheltuiala cu impozitul pe profit este 14.400 (dacă ignorăm ajustarea soldului vechi) sau 13.760 (dacă ajustăm soldul vechi). Textul cere "Reconcilierea numerică dintre cheltuiala cu impozitul... și valoarea obținută prin înmulțirea rezultatului contabil".

Reconciliere:
$$Rezultat Contabil \times 16\% = 16.000$$
$$- \text{Efect venituri neimpozabile (Dividende)} (10.000 \times 16\%) = -1.600$$
$$= \text{Cheltuiala cu impozitul pe profit} = 14.400$$

*(Notă: Soldul inițial de 640 este o ajustare de stoc, nu de flux curent, de aceea în reconcilierea standard de flux se folosește cheltuiala generată în anul curent).*

### Subiectul 3: Active deținute pentru vânzare

1.  **Situația la 31.12.N+1 (înainte de reclasificare):**
    *   Cost (31.12.N-1): 22.000.
    *   Amortizare N (an 1): $(22.000 - 2.000) / 5 = 4.000$. VCN = 18.000.
    *   *Modificare estimare 31.12.N+1:*
        *   Activul a mai fost folosit un an (N+1). Înainte de schimbare, amortizarea pe N+1 ar fi fost tot 4.000.
        *   VCN înainte de recalibrare la finele N+1 = 14.000.
        *   Managerul schimbă durata la 6 ani *din acest moment* și reziduala la 500. Aceasta afectează calculul *viitor* (începând cu N+2).
2.  **Reclasificare la 01.07.N+2:**
    *   Amortizare pe 6 luni în N+2 (până la 1 iulie):
    *   VCN la 01.01.N+2 = 14.000. Noua reziduală = 500. Durata rămasă = 6 ani.
    *   Amortizare anuală nouă = $(14.000 - 500) / 6 = 2.250$.
    *   Amortizare semestrială (Ian-Iun) = $1.125$.
    *   VCN la 01.07.N+2 = $14.000 - 1.125 = 12.875$.
    *   **Evaluare la reclasificare (IFRS 5):**
        *   Minimul dintre VCN (12.875) și Valoarea Justă minus costuri vânzare ($13.400 - 400 = 13.000$).
        *   Minim = 12.875.
        *   Se reclasifică la 12.875. Nu se înregistrează câștig.
3.  **Evaluare la 31.12.N+2:**
    *   Activul nu se mai amortizează.
    *   Valoarea justă minus costuri = 13.800.
    *   VCN actuală = 12.875.
    *   Se poate recunoaște un câștig din reluarea pierderii anterioare? Nu avem pierdere de valoare înregistrată la transfer. IFRS 5 permite creșterea valorii până la cumularea pierderilor din depreciere anterioare, dar aici valoarea a crescut peste valoarea contabilă.
    *   Conform IFRS 5, un câștig pentru o creștere ulterioară a valorii juste minus costurile de vânzare este recunoscut, dar nu peste pierderea cumulată din depreciere (inclusiv cea dinainte de reclasificare). Aici nu am avut pierdere la reclasificare. Deci rămâne la 12.875?
    *   *Corecție:* Dacă valoarea justă crește, nu putem reevalua activul peste costul istoric amortizat. Totuși, IFRS 5 specifică recunoașterea câștigului, dar limitat. În acest caz specific, deoarece la transfer VCN < VJ netă, am transferat la VCN. Acum VJ netă crește și mai mult. Nu înregistrăm profitul nerealizat.
